💡 Tips

  • TIFF는 인쇄·의료·스캔 이미지에 많이 사용되는 고화질 포맷입니다.
  • JPG 변환 시 품질을 90 이상으로 설정하면 화질 손실을 최소화할 수 있습니다.
  • 투명도가 있는 TIFF는 PNG로 변환하면 투명도가 유지됩니다.

자주 묻는 질문

TIFF 파일이란 무엇인가요?

TIFF(Tagged Image File Format)는 인쇄·출판·의료 분야에서 사용하는 고화질 무손실 이미지 형식입니다. 파일 크기가 크지만 화질이 매우 우수합니다.

여러 TIFF 파일을 한 번에 변환할 수 있나요?

네. 여러 파일을 선택하면 일괄 변환 후 ZIP으로 다운로드됩니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

자주 묻는 질문

OG 이미지란 무엇인가요?

OG 이미지(Open Graph Image)는 링크를 SNS나 메신저에 공유할 때 표시되는 미리보기 이미지입니다. Facebook, Twitter, Kakao, Slack 등에서 자동으로 표시됩니다.

OG 이미지 권장 크기는?

표준 크기는 1200×630픽셀(1.91:1 비율)입니다. 이 도구는 해당 크기로 자동 생성합니다.

생성된 이미지를 어떻게 사용하나요?

PNG로 다운로드 후 웹서버에 업로드하고, HTML head 태그에 <meta property="og:image" content="이미지URL">을 추가하면 됩니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

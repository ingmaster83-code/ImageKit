🖼️ 이미지킷이란?

이미지킷은 이미지 압축, 리사이즈, 형식 변환, 회전, 자르기, 워터마크 추가 등 다양한 이미지 작업을 브라우저에서 직접 처리하는 무료 온라인 도구 모음입니다.

회원가입 없이 바로 사용할 수 있으며, 업로드된 파일이 서버에 저장되지 않아 개인 정보와 이미지를 안전하게 보호합니다.

✅ 서비스 특징

  • 모든 이미지 처리가 브라우저 내 Canvas API로 이루어져 파일이 서버에 전송되지 않음
  • 회원가입 없이 즉시 사용 가능, 100% 무료
  • 이미지 압축·리사이즈·변환·자르기·회전·워터마크·Base64 변환 등 11가지 도구
  • JPG, PNG, WebP, GIF 등 주요 이미지 형식 지원
  • PC·태블릿·스마트폰 모든 기기에서 편리하게 이용 가능

🛠️ 제공 도구

  • 이미지 압축 — JPG/PNG 파일 용량 줄이기
  • 이미지 리사이즈 — 픽셀 또는 비율로 크기 조정
  • 이미지 회전/뒤집기 — 90°/180°/270° 회전, 좌우/상하 반전
  • 이미지 자르기 — 자유 크롭 또는 비율 크롭
  • JPG → PNG / PNG → JPG 변환
  • 이미지 → WebP 변환 — 용량 절감 최적화
  • WebP → JPG/PNG 변환
  • 이미지 → Base64 인코딩
  • 워터마크 추가 — 텍스트 워터마크 삽입
  • 이미지 정보 확인 — 해상도, 파일 크기, 형식 등

💡 Tips

  • ICO 파일에는 여러 크기의 이미지가 포함되어 있으며, 브라우저가 선택한 크기로 변환됩니다.
  • PNG는 투명도를 지원하므로 아이콘의 투명 배경이 보존됩니다.
  • PNG를 ICO로 변환하려면 PNG → ICO 도구를 사용하세요.

자주 묻는 질문

ICO 파일이란 무엇인가요?

ICO는 Windows 아이콘 형식으로, 웹사이트 파비콘·앱 아이콘에 사용됩니다. 여러 크기의 이미지를 하나의 파일에 포함할 수 있습니다.

PNG → ICO로 변환도 가능한가요?

네. WooaImage의 PNG → ICO 도구를 사용하면 PNG를 ICO로 변환할 수 있습니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

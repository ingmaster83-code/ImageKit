💡 Tips

  • Base64로 변환하면 HTML·CSS에 이미지를 직접 삽입할 수 있습니다.
  • 파일 크기가 약 33% 커지므로 작은 아이콘에 주로 사용합니다.
  • data:image/png;base64,... 형식으로 img src에 바로 붙여 사용하세요.

자주 묻는 질문

Base64로 변환하면 무엇에 쓰나요?

HTML이나 CSS에 이미지를 직접 삽입하거나 API 전송 시 이미지를 텍스트로 인코딩할 때 사용합니다.

변환된 Base64 문자열이 매우 긴데 정상인가요?

네, Base64는 원본보다 약 33% 크기가 증가하므로 문자열이 길어집니다.

Base64를 다시 이미지로 변환할 수 있나요?

이 도구는 이미지→Base64 방향만 지원합니다. 역변환은 개발자 도구나 온라인 디코더를 이용하세요.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.
